## БАЗОВЫЕ КОМАНДЫ
- ### при создании нового проекта, необходимо использовать команду 
> django-admin startproject nameproject

вместо nameproject можно использовать любое имя для вашего проекта

---

- ### при создании нового приложения для проекта, необходимо использовать команду

***необходимо при использовании команды перейти в директорию с проектом***

> python manage.py startapp nameapp

вместо nameapp можно использовать любое имя для вашего приложения

---

- ### миграции к базе данных

***необходимо при использовании команды перейти в директорию с проектом***

> python manage.py migrate 

команда применяет миграции к базе данных, которая указана в файле settings.py. Миграции - это способ создавать и изменять таблицы в базе данных на основе определенных классов моделей в приложениях Django. Эта команда также применяет миграции по умолчанию для встроенных приложений Django, таких как admin, auth, contenttypes и sessions.

---

- ### создание суперпользователя для админки
  
> python manage.py createsuperuser

В ходе выполнения команды появится приглашение ввести имя пользователя, пароль и емейл (они не отображаются во время ввода – будьте внимательны). Учетная запись суперпользователя готова, запустите сервер и авторизуйтесь.
